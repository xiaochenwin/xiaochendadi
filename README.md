# 围巾大帝萧尘

> 一条围巾，见证万古；一颗真心，守护永恒。

## 📖 作品简介

《围巾大帝萧尘》是一部古典仙侠玄幻小说，讲述了八荒第一大帝萧尘与紫曦仙子之间跨越万古的爱情故事，以及萧尘从大帝陨落到重修崛起，最终超脱天道、守护苍生的传奇经历。

## 🎯 核心主题

- **爱与守护**：以围巾为象征，诠释爱的永恒与守护的不朽
- **初心不改**：从大帝到凡人，始终坚守初心
- **大道超脱**：不断突破，追求更高境界
- **传承延续**：道统传承，万世不衰

## 🎭 核心物品

### 围巾

- **来历**：紫曦仙子亲手以万年玄蚕丝织就，萧尘证道大帝之日炼制七七四十九日而成
- **特点**：内蕴帝威，可护心神万劫不灭；蕴含紫曦残魂，与萧尘心意相通
- **象征**：爱与守护的永恒象征，贯穿全书的核心意象

### 帝树

- **来历**：萧尘与紫曦隐居时种下
- **成长**：从幼苗长成参天大树，根系扎入虚空，枝叶连接天地
- **作用**：护佑人间，灵气充裕，成为圣物

## 💫 核心情节

### 大帝陨落（第1-30章）

萧尘作为八荒第一大帝，为救紫曦，斩尽魔族，违逆天道，被贬入凡尘。

### 重修崛起（第31-80章）

萧尘从凡人重修，重回大帝之境，斩杀魔祖，再次守护人间。

### 天外天劫（第81-150章）

萧尘与紫曦踏入天界，斩混沌魔神，参悟大道本源，超脱天道。

### 万古传说（第151-220章）

萧尘与紫曦回归大道本源，后人传承道统，守护宇宙万世不衰。

### 归真之路（第201-300章）

萧尘与紫曦达到归真境，真正圆满，永恒相伴，万古不朽。
